Precision rescue intelligence monitoring system (PRIAMS)

The Precision Rescue Ambulance Intelligence Monitoring System is designed to expedite the movement of ambulances through congested traffic. By leveraging GPS technology, the system identifies the location of approaching ambulances and communicates this information to nearby traffic signals. When an ambulance is within a specified distance (e.g., 2 kilometers) of a traffic light, the signal is automatically turned red to clear the intersection, allowing the ambulance to proceed without delay.

My role

Designed and Executed Wiring Setup

Developed the primary code for GPS tracking using the TinyGPS++ library to accurately monitor ambulance locations. Established a reliable communication link between ambulances and traffic signal control systems to prioritize emergency vehicles. DNA based encryption system

This project implements a novel encryption system that leverages DNA sequences for enhanced data security. Unlike traditional methods that rely on fixed DNA sequences, this system generates a unique 128-bit DNA sequence for each encryption cycle. The dynamic generation of DNA sequences makes the system more secure and harder to break, addressing the limitations of earlier encryption systems that used fixed sequences.
